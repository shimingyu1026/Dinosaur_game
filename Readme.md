# Dinosaur_Game
## 使用平台
Windows11 22H2

## C语言环境
Mingw x86_64-w64-mingw32
gcc version 12.2.0

## 编译选项

IDE Vscode
编译指令 gcc fileName -fexec-charset=gbk -o fileNameWithoutExt -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_ttf

## 第三方库

SDL2
SDL2_image
SDL2_ttf ~~好像并没有用到~~

## 游戏方式

空格、上键跳跃

下键下蹲

起跳过程中按下键直接下落，下蹲时按跳跃无反应
